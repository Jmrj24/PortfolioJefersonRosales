Sobre Mí

Soy Desarrollador Backend enfocado en Java y Spring Boot. Mi prioridad es construir aplicaciones bien hechas: ordenadas, seguras y fáciles de mantener.

Me motiva entender a fondo las herramientas que utilizo, profundizando en Spring Security, JWT y Docker para implementar arquitecturas robustas desde la base. Busco un equipo donde pueda aportar calidad técnica y seguir creciendo en proyectos reales.

Stack Tecnológico

Core: Java 17, Spring Boot 3, Spring Security 6.
Infra: Docker, Git, MySQL.
Arquitectura: Microservicios, API REST, JWT, Circuit Breaker.

Sistema de Gestión Académica

API RESTful diseñada para la administración escolar con un enfoque prioritario en la Seguridad.

  • Seguridad Stateless: Implementación manual de JWT y filtros personalizados con Spring Security 6.
  • Integridad: Transacciones atómicas (@Transactional) para registros complejos.
  • Deploy: Contenerizado con Docker y documentado con Swagger UI.

Gestión de Tareas (Microservicios)

Sistema distribuido optimizado para alta disponibilidad y tolerancia a fallos.
